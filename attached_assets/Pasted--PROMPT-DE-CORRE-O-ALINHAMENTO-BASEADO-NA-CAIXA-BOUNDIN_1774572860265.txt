### PROMPT DE CORREÇÃO: ALINHAMENTO BASEADO NA CAIXA (BOUNDING BOX)

**Atua como um Desenvolvedor Front-end Sénior.**
Temos um problema conceitual na forma como o alinhamento de texto (Left, Center, Right) está a ser calculado na função `generateCertificatePdf` do arquivo `client/src/pages/dashboard/capacitacao.tsx`.

**O PROBLEMA:**
Atualmente, o cálculo do alinhamento usa a largura da própria linha do texto (`Math.max(...lineWidths)`). Isso faz com que o texto centralizado fique estático no ponto de origem do Drag & Drop. O utilizador espera o comportamento de um editor profissional: a "caixa" arrastável na tela define as margens (bounding box), e o texto deve alinhar-se **dentro dos limites dessa caixa**.

**A SOLUÇÃO:**
1. A largura da "caixa invisível" (Bounding Box Width) deve ser definida pelo próprio tamanho do texto convertido, OU por uma largura total da página menos margens, caso a intenção seja centralizar na folha.
Como a implementação atual do `NativeDraggableBlock` não possui "Resize" (redimensionamento de largura) manual, a melhor forma de centralizar num certificado é usar uma **Largura Definida (Fixed Width)** para elementos centralizados, ou basear-se no `pdfWidth`.

**ALTERAÇÃO OBRIGATÓRIA NO CÁLCULO DE ALINHAMENTO (DENTRO DE `generateCertificatePdf`):**

Substitui a lógica de `lineX` pela seguinte matemática:

```javascript
// Usa a largura máxima configurada para o bloco (ou 80% do PDF como fallback seguro)
const blockMaxWidth = pdfWidth * 0.8; 

// Se for centralizado, ignoramos a coordenada X arrastada e usamos o centro real da página
// Isso garante um alinhamento perfeito de títulos e nomes no meio do diploma!
lines.forEach((line, index) => {
  const lineWidth = embedFont.widthOfTextAtSize(line, safeSize);
  
  let lineX = finalX; // Padrão: Esquerda

  if (block.align === 'center') {
    // Modo 1: Centraliza exatamente no meio da PÁGINA (Padrão de Certificados)
    // Se quiseres que centralize no bloco arrastado, usa: lineX = finalX + (blockMaxWidth - lineWidth) / 2;
    // O mais seguro para diplomas é forçar o centro da página:
    lineX = (pdfWidth - lineWidth) / 2; 
  } else if (block.align === 'right') {
    // Alinha à direita com base na margem configurada
    lineX = finalX + (blockMaxWidth - lineWidth);
  }

  const lineY = finalY - (index * safeSize * 1.2);

  page.drawText(line, {
    x: Math.max(0, Math.min(lineX, pdfWidth - 10)),
    y: Math.max(0, Math.min(lineY, pdfHeight - safeSize)),
    size: Math.max(6, safeSize),
    font: embedFont,
    color: rgb(0, 0, 0),
  });
});
