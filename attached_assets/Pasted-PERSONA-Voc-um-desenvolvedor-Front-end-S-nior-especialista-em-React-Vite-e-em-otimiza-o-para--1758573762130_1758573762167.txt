PERSONA: Você é um desenvolvedor Front-end Sênior, especialista em React, Vite e em otimização para SEO e redes sociais (Social Media Optimization).

OBJETIVO PRINCIPAL: Configurar o projeto para exibir meta tags de preview dinâmicas para a rota específica /gomatoken. Queremos que, ao compartilhar o link dessa página, apareça uma imagem, título e descrição personalizados.

CONTEXTO TÉCNICO:

Framework: React com Vite.

Estrutura: É uma Aplicação de Página Única (SPA).

Arquivo Principal: client/src/main.tsx

Componente da Rota Alvo: client/src/pages/gomatoken.tsx

Hospedagem: O deploy é feito no Netlify e a funcionalidade de Prerendering já foi ativada.

Por favor, execute as seguintes tarefas, passo a passo:

PASSO 1: Instalar a dependência para gerenciar o <head>

No terminal (Shell), execute o comando abaixo para instalar a biblioteca react-helmet-async, que nos permitirá gerenciar as meta tags de forma dinâmica dentro dos componentes React.

Bash

npm install react-helmet-async
PASSO 2: Configurar o HelmetProvider na raiz da aplicação

Modifique o arquivo client/src/main.tsx para envolver toda a aplicação com o HelmetProvider. Isso é necessário para que o Helmet funcione em qualquer lugar do projeto.

Arquivo a ser modificado: client/src/main.tsx

Instruções:

Importe o HelmetProvider de react-helmet-async.

Envolva o componente <App /> com o <HelmetProvider>.

O código final deve ficar semelhante a este:

TypeScript

import React from 'react';
import ReactDOM from 'react-dom/client';
import { HelmetProvider } from 'react-helmet-async';
import App from './App';
import './index.css';

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <HelmetProvider>
      <App />
    </HelmetProvider>
  </React.StrictMode>
);
