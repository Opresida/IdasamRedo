Prompt Mestre para a Recriação do Site do Idasam (Versão Final)
Objetivo: Gerar o código completo para o novo site do Instituto de Desenvolvimento Ambiental e Social da Amazônia (Idasam), utilizando os componentes React especificados, com todo o conteúdo, imagens e identidade visual definidos.

Estrutura do Projeto:
O projeto deve ser compatível com a estrutura shadcn, utilizando Tailwind CSS e TypeScript.

Tema Geral e Paleta de Cores:
O design deve ser moderno, limpo e profissional, com uma forte identidade visual amazônica.

Paleta de Cores Principal:

Verde Escuro (floresta): #1A4D2E

Terracota (terra): #C86A3B

Bege (areia/fundo): #F5EFE6

Azul-petróleo (rios/destaque): #008080

Tipografia: Utilizar uma fonte limpa e legível (Ex: Inter, Lato).

Ícones: Utilizar a biblioteca lucide-react.
